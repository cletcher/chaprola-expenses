Export Pipeline

How it works: When you click Export, the proxy server calls Chaprola's /export-report endpoint, which runs the compiled .PR program against the ledger data and generates the output file. The file is stored temporarily in S3, and you receive a presigned download URL.
Step Chaprola Endpoint Detail
1 /compile Compile Chaprola source (.CS) to bytecode (.PR)
2 /publish Make the program available for public access
3 /export-report Run program and save output to S3 in requested format
4 /download Get presigned URL for the generated file
