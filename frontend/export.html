How it works

This page issues a single POST /query to api.chaprola.org with the date-range filter and serializes the result in the browser. CSV and JSON are built client-side and delivered via a blob URL — no server conversion, no download URL round-trip. The dedicated DETAIL.CS program (a parameterized detail report) is still available for direct GET /report?name=DETAIL&category=Travel consumption.
